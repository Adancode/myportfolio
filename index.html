Adventures Abroad: Mapael.js & API's

front end

Adventures Abroad contains a map of the world generated by Mapael.js. The map is color coded to show how long an American can stay in a country upon arrival. Hovering over a country reveals a tooltip with the name of the country as well as the country's population. Clicking on a country will cause several divs to appear below the map. The first div gives users access to a Firebase database so they can view and leave comments about the country. The second div accesses the New York Times API to pull up recent news stories about that country, that way users can have the latest information, in case current events could affect their travel plans. Finally, the last div accesses the Flickr API with a search run for that country, to give the user a glimpse into that country's culture.

This project required the manual input of a lot of information. I had to research every country's laws regarding Americans' ability to visit it, and then create the color-coded legend to make sense of the data at a glance.

This project could be improved by adding country-specific visa data to the tooltip, but that would take quite a bit of time. Another improvement that could be made is adding a dropdown menu that allows the user to switch the country for which the map provides data for, so for example the user could see visa requirements for Russians, Chinese, or Filipinos, by simply making a selection in the dropdown menu.

I really enjoyed this project, as it has real-world applicability.
